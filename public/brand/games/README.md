# Game logo assets

Standard can render approved game logo SVGs from this directory. If a logo file
is absent, fails to load, or the game is unknown, the UI falls back to the
existing stylized letter/gradient badge.

Expected filenames:

- `valorant.svg`
- `cs2.svg`
- `fortnite.svg`
- `apex-legends.svg`
- `call-of-duty.svg`
- `league-of-legends.svg`
- `escape-from-tarkov.svg`
- `rust.svg`

Guidelines:

- SVG is preferred.
- Use a transparent background.
- Keep files optimized and suitable for small badge rendering.
- Do not hotlink remote logos.
- Do not add third-party logo/icon packages for these assets.
- Only commit game logo files when the owner has explicitly provided or
  approved them for use.
- Treat all game names and logos as trademarks of their respective owners.
